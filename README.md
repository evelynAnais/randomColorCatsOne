# RandomColorCats

Simple web app for random cat facts and colors

## Description

Used an API that gives 5 random cat facts a day. Made it so that when you click the button you get a random fact and a random color in the background.

## Getting Started

### Dependencies

* axios

### Installing
 
```bash 
npm install
```
